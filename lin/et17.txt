في بايثون ، إذا تم استخدام عبارة else elif في اتخاذ القرار.

إذا البيان
إذا كان البيان هو أبسط بيان لاتخاذ القرار. يتم استخدامه لتحديد ما إذا كان سيتم تنفيذ عبارة معينة أو مجموعة من العبارات أم لا ، أي إذا كان شرط معين صحيحًا ، فسيتم تنفيذ كتلة من البيان وإلا فلن يتم تنفيذ ذلك.

بناء الجملة : 

إذا الشرط :
   # جمل لتنفيذها إذا
   # الشرط صحيح
هنا ، ستكون الحالة بعد التقييم إما صحيحة أو خاطئة. إذا كانت العبارة تقبل القيم المنطقية - إذا كانت القيمة صحيحة ، فسيتم تنفيذ كتلة العبارات الموجودة أسفلها وإلا فلن يتم ذلك. يمكننا استخدام الشرط مع القوسين '(' ')' أيضًا. 

كما نعلم ، يستخدم بيثون المسافة البادئة لتحديد كتلة. لذلك سيتم تحديد الكتلة الموجودة أسفل عبارة if كما هو موضح في المثال أدناه:

إذا كان غير ذلك
تخبرنا عبارة if وحدها أنه إذا كان الشرط صحيحًا ، فسيتم تنفيذ مجموعة من العبارات وإذا كان الشرط خاطئًا ، فلن يفعل ذلك. ولكن ماذا لو أردنا القيام بشيء آخر إذا كانت الحالة خاطئة. هنا يأتي بيان آخر . يمكننا استخدام تعليمة else مع عبارة if لتنفيذ كتلة من التعليمات البرمجية عندما يكون الشرط خاطئًا.

    # الشرط خاطئ

# python program to illustrate If else statement
#!/usr/bin/python
 
i = 20
if (i < 15):
    print("i is smaller than 15")
    print("i'm in if Block")
else:
    print("i is greater than 15")
    print("i'm in else Block")
print("i'm not in if and not in else Block")
انتاج: 

أنا أكبر من 15
أنا في آخر بلوك

مثال 2: بايثون إذا كان الأمر كذلك في قائمة الفهم

# Explicit function
def digitSum(n):
    dsum = 0
    for ele in str(n):
        dsum += int(ele)
    return dsum
 
 
# Initializing list
List = [367, 111, 562, 945, 6726, 873]
 
# Using the function on odd elements of the list
newList = [digitSum(i) for i in List if i & 1]
 
# Displaying new list
print(newList)

مثال: Python Nested if

# python program to illustrate nested If statement
#!/usr/bin/python
i = 10
if (i == 10):
   
    #  First if statement
    if (i < 15):
        print("i is smaller than 15")
         
    # Nested - if statement
    # Will only be executed if statement above
    # it is true
    if (i < 12):
        print("i is smaller than 12 too")
    else:
        print("i is greater than 15")
